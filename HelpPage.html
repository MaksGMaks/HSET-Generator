High Speed Traffic Generator

Welcome to the official manual. This tool is designed for high-performance network stress testing using DPDK and PF_RING technologies.

General Look

The application interface is organized into 4 logical zones:

  • Parameters Block: Configuration of network interfaces and packet headers.
  • Info Block: Real-time statistics and counters.
  • Graph Block: Visual representation of throughput (bps/BPS).
  • Control Block: Main execution controls (Start, Stop, Pause).

Control Block

Located at the base of the dashboard, this block manages the generator lifecycle.

  • Start/Stop: Toggle traffic generation. Note that parameters are locked while active.
  • Pause/Resume: Temporarily halt transmission without resetting counters.
  • Time Selector: 00:00:00 for continuous flow, or set a specific countdown.

Parameters

Interface: The network device (NIC) used for output.

Packet Size: 16B to 64KB. Smaller packets increase CPU overhead.

Burst Size: (DPDK/ZC) Number of packets sent per DMA transaction.

Copies / Sent: Termination limits. Set to 0 for infinite.

Generating Modes

PF_RING_STANDARD Uses default PF_RING sockets. Requires pfring.ko module.

PF_RING_ZC Zero Copy mode. Bypasses kernel. Requires specific ZC drivers.

DPDK Direct hardware access. Requires HugePages and DPDK environment.

Important Note on Speed

Preferred speed is a target. Real-world output may be lower due to hardware limits, small packet overhead, or kernel latency.
